\section{Procédures de Métrologie et de Calibration (Norme ISO 230-1)}

\subsection{Principe et Méthodologie}
La norme ISO 230-1 définit les procédures standardisées d'essai des machines-outils à CN afin de quantifier leurs performances géométriques. Les grandeurs mesurées sont :
\begin{itemize}
    \item \textbf{Précision de Positionnement ($P$)} : Étendue totale des erreurs de position sur une série de cycles.
    \item \textbf{Répétabilité ($R_p$)} : Dispersion des positions atteintes lors de mouvements unidirectionnels répétés.
    \item \textbf{Hystérésis ($B$)} : Erreur systématique due au jeu d'inversion (Backlash), mesurée lors de mouvements bidirectionnels.
\end{itemize}

\subsection{Protocole de Mesure (Méthodologie Adoptée)}
L'instrument de mesure utilisé est un comparateur à cadran numérique de résolution $1\,\mu m$ (0{,}001 mm), fixé sur un pied magnétique indépendant.

\textbf{Protocole pour chaque axe linéaire (X, Y, Z) :}
\begin{enumerate}
    \item L'axe est amené à l'origine machine (Homing).
    \item La pointe du comparateur est positionnée contre une référence (face plane de l'étau).
    \item Le zéro du comparateur est remis à zéro (offset G92).
    \item Une commande de déplacement de $\pm D$ (distance de test = 50 mm) est émise depuis l'IHM.
    \item L'indication du comparateur est relevée manuellement. L'opération est répétée $n = 20$ fois consécutives.
\end{enumerate}

\section{Analyse Statistique de la Capabilité Machine ($C_{pk}$)}

\subsection{Théorie de la Capabilité (ISO 22514)}
La capabilité machine $C_{pk}$ est un indice qui mesure si le processus (la machine) est statistiquement capable de produire des pièces dans les limites de tolérance. Un $C_{pk} \geq 1{,}33$ est requis pour une production de qualité industrielle.

\begin{equation}
    C_{pk} = \min\left(\frac{USL - \bar{x}}{3\sigma},\, \frac{\bar{x} - LSL}{3\sigma}\right)
\end{equation}
Où :
\begin{itemize}
    \item $USL / LSL$ = Limite Supérieure / Inférieure de Spécification
    \item $\bar{x}$ = Moyenne des mesures
    \item $\sigma$ = Écart-type (Déviation Standard)
\end{itemize}

\subsection{Calcul du $C_{pk}$ pour l'Axe Y (Cas Critique)}
Pour l'axe Y (le plus défavorable), avec les données du tableau~\ref{tab:precision_tests} :
\begin{itemize}
    \item $\bar{x} = -29\,\mu m$ (erreur moyenne de position)
    \item $\sigma = R_p / (2 \times 3) = 23 / 6 \approx 3{,}83\,\mu m$ (en supposant une loi normale)
    \item $USL = +50\,\mu m$, $LSL = -50\,\mu m$ (tolérance bilatérale)
\end{itemize}
\begin{equation}
    C_{pk} = \min\left(\frac{50 - (-29)}{3 \times 3{,}83},\, \frac{(-29) - (-50)}{3 \times 3{,}83}\right) = \min\left(\frac{79}{11{,}5},\, \frac{21}{11{,}5}\right) = \min(6{,}87,\, 1{,}83) = \boxed{1{,}83}
\end{equation}
Un $C_{pk} = 1{,}83$ confirme que l'axe Y, malgré son erreur systématique de centrage, reste statistiquement capable pour la production de pièces nécessitant une tolérance de $\pm 50\,\mu m$.

\section{Tests d'Usinage (Validation Fonctionnelle Pratique)}

\subsection{Test 1 : Gravure 2.5D (Validation Axes Linéaires)}
La gravure d'une plaque de HDPE (Polyéthylène Haute Densité, $100 \times 80 \times 10\text{ mm}$) avec des lettres «ENI-ABT CNC 5 AXES» permet de valider les axes X et Y en gravure 2.5D.
\begin{itemize}
    \item Outil : Fraise $3\text{ mm}$ (1 dent) / Aluminium
    \item Profondeur : $a_p = -1\text{ mm}$
    \item Avance : $F = 300\text{ mm/min}$
    \item Résultat : Rugosité de fond de gravure $Ra \approx 3{,}2\,\mu m$ (mesurée par Rugotest tactile). Lettres parfaitement lisibles.
\end{itemize}

\subsection{Test 2 : Poche 3+2 (Validation Positionnement Axe A+B)}
Usinage d'une poche inclinée à $45°$ sur face supérieure d'un cube de résine Cibatool ($50 \times 50 \times 40\text{ mm}$).
\begin{itemize}
    \item Stratégie : \textit{Face milling} en 3+2 (A = 45°, B = 0°) — axes A et B bloqués.
    \item Fraise $6\text{ mm}$ plate / 2 dents.
    \item Résultat : Mesure au pied à coulisse de l'angle de la face usinée : $\mathbf{44{,}93° \pm 0{,}07°}$.
    \item Conformité : Erreur angulaire $< 0{,}1°$ — CONFORME.
\end{itemize}

\subsection{Test 3 : Usinage Continu 5 Axes (Validation Cinématique)}
Usinage d'un dôme hémisphérique ($R = 25\text{ mm}$) sur aluminium AW-2017A par interpolation 5 axes simultanées via G-Code exporté depuis Fusion 360.
\begin{itemize}
    \item 8~932 lignes G-Code, durée $\approx 1\text{h}23\text{min}$
    \item Résultat : Le profil mesuré par report sur gabarit circulaire correspond à la géométrie théorique avec écart maximum $< 0{,}08\text{ mm}$.
    \item Rugosité de surface $Ra \approx 1{,}6\,\mu m$ (surfaces d'aspect «poli-miroir») — validé.
\end{itemize}

\subsection{Analyse de Rentabilité (ROI) pour une PME Locale}
Si la technologie développée dans ce PFE était mise en échange commercial (service d'usinage), le retour sur investissement peut être calculé.
\begin{itemize}
    \item Investissement initial : $I_0 = 521~500 \text{ FCFA}$
    \item Taux horaire facturable (Usinage prototypes aluminium) : $H_f = 5~000 \text{ FCFA/h}$
    \item Temps d'usinage productif estimé : $T_{utile} = 800 \text{ h/an}$ (soit $4\text{ h}$ par jour ouvré / 200 jours)
    \item Revenus annuels : $R_a = H_f \times T_{utile} = 5~000 \times 800 = 4~000~000 \text{ FCFA/an}$
\end{itemize}
\begin{equation}
    \text{Durée de Retour sur Investissement (ROI)} = \frac{I_0}{R_a} = \frac{521~500}{4~000~000} \approx \boxed{1{,}56\text{ mois}}
\end{equation}
Un retour sur investissement en moins de 2 mois confirme la viabilité économique exceptionnelle de la solution open-source pour les PME locales.

\section{Conclusion Générale du Projet}

Ce projet de fin d'études a permis de concevoir, réaliser, programmer et valider une \textbf{mini-fraiseuse CNC 5 axes à architecture Trunnion}. Partant d'un besoin identifié (l'inexistence de systèmes CNC multiaxes locaux accessibles à l'ENI-ABT), la démarche rigoureuse d'ingénierie adoptée a conduit à des résultats quantifiés et vérifiés.

\paragraph{Bilan des performances atteintes :}
\begin{itemize}
    \item Précision de positionnement : $\leq \pm 0{,}05\text{ mm}$ sur les 5 axes (cible : $\pm 0{,}05\text{ mm}$) $\checkmark$
    \item Capabilité machine $C_{pk} = 1{,}83$ sur l'axe le plus critique (cible $> 1{,}33$) $\checkmark$
    \item Latence de communication IHM-Machine : $< 3\text{ ms}$ (cible $< 5\text{ ms}$) $\checkmark$
    \item Volume de travail : $200 \times 200 \times 150\text{ mm}^3$ (cible atteinte) $\checkmark$
    \item Coût total : $521~500\text{ FCFA}$ soit $-92\%$ vs solution commerciale $\checkmark$
    \item Usinage 5 axes continus validé (dôme R25mm, $Ra = 1{,}6\,\mu m$) $\checkmark$
\end{itemize}

\paragraph{Perspectives d'amélioration :}
\begin{enumerate}
    \item Intégration d'un \textbf{Palpeur de mesure} (Renishaw-like à déclencheur piézoélectrique) pour le Homing automatique du TCP.
    \item Migration vers l'ESP32-S3 (RISC-V) avec DSP matériel pour l'implémentation de S-Curves à 7 phases plus précises.
    \item Ajout d'un système de \textbf{4ème axe continu (broche orientable)} pour passer à une machine de type 6 axes.
    \item Développement d'un module FAO intégré à l'IHM Flutter (post-processeur Trunnion natif).
\end{enumerate}
